📝 OverviewThis repository contains a comprehensive forensic analysis of a sophisticated phishing campaign intercepted on Friday, 4 August 2023. The attack impersonated the global sporting-goods retailer Decathlon, using a €1,000 gift voucher lure to target German-speaking recipients.  The investigation covers the full attack lifecycle, from header forensics and HTML source inspection to reconstructing the malicious redirect chain.  🔍 Key FindingsThe analysis uncovered 8 distinct Indicators of Compromise (IoCs):  Authentication Failures: A "Triple Failure" where SPF, DKIM, and DMARC all failed simultaneously.  Infrastructure: The email originated from granigo.art (IP: 80.96.157.210).  Malicious Redirection: Use of an obfuscated URL on bsq2.firiri.shop disguised as a legitimate "unsubscribe" link.  Social Engineering: High-pressure tactics leveraging brand impersonation, greed (prize lure), and professional HTML layouts.  🛠️ Tools UsedMicrosoft Outlook: For rendered email and header inspection.  Email Header Analyser: For SMTP hop and authentication record tracing.  HTML Source Inspector: For identifying hidden malicious redirect URLs.  🛡️ MITRE ATT&CK® Techniques MappedTechnique IDNameDescriptionT1566.001Phishing: Spearphishing Attachment/LinkSpoofed sender display names and authentication bypass.T1566.002Phishing: Spearphishing LinkUse of attacker-controlled redirect domains.T1598Search Victim-Owned WebsitesSocial engineering through prize/voucher lures.T1027Obfuscated Files or InformationExtremely long, encoded URLs to evade detection.📂 Analysis PhasesHeader Analysis: Identifying the true sending server and authentication gaps.
